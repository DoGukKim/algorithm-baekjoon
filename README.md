# 🧮 백준 알고리즘 문제 풀이

백준 온라인 저지(BOJ) 알고리즘 문제 풀이를 정리한 저장소입니다.

## 📁 프로젝트 구조

```
algorithm-baekjoon/
├── bronze/          # 브론즈 난이도 문제
├── sliver/          # 실버 난이도 문제
├── gold/            # 골드 난이도 문제
├── index.js         # JavaScript 템플릿
├── index.py         # Python 템플릿
└── input.txt        # 로컬 테스트용 입력 파일
```

## 🛠️ 사용 언어

| 언어                 | 확장자 |
| -------------------- | ------ |
| Python               | `.py`  |
| JavaScript (Node.js) | `.js`  |

## 📝 풀이 형식

각 문제 풀이 파일에는 다음 정보가 포함되어 있습니다:

- **시간 복잡도** (TimeComplexity)
- **공간 복잡도** (SpaceComplexity)
- **풀이 코드**

## 🚀 로컬 실행 방법

### 1. 입력 데이터 준비

`input.txt` 파일에 테스트할 입력 데이터를 작성합니다.

### 2. 실행

**Python:**

```bash
python <파일경로>.py
```

**JavaScript (Node.js):**

```bash
node <파일경로>.js
```
